# Stateless-AI-Chatbot
A basic stateless ai chatbot using Google API.

## Directory Structure

```
stateless-ai-chatbot/
├── .env
├── package.json
├── server/
│   └── server.js
└── src/
    └── App.js
    └── index.js
    └── styles.css
    └── components/
        ├── ChatInput.js
        ├──ChatMessage.js
        ├── ChatWindow.js
        ├── Sidebar.js
        └── UI/
            ├── Button.js
            └── Input.js
```
